Planning d’apprentissage Laravel + Vue + Inertia + Tailwind

Durée : 2 mois — Objectif : Maîtrise complète des bases Laravel, Vue.js, Inertia.js, Tailwind CSS

Module 1 : Laravel Basics

Semaine 1

  • Installation de Laravel, configuration de l'environnement (sail, XAMPP ou Laragon)
  • Architecture MVC de Laravel
  • Les routes : web.php et api.php
  • Contrôleurs, vues (Blade) et modèles
  • Migrations, factories, seeders
  • Commandes artisan les plus utiles

Module 2 : Authentification, Middleware et Sécurité

Semaine 2

  • Laravel Breeze (ou Jetstream/Inertia)
  • Pages Login / Register personnalisées
  • Middleware d’authentification et de rôles
  • Validation côté serveur (Request classes)
  • Hashing des mots de passe et sécurité CSRF
  • Logout, redirection selon rôles, accès protégé

Module 3 : Vue.js Basics + Intégration Inertia

Semaine 3

  • Vue 3 composition API avec script setup
  • Création de composants Vue réutilisables
  • Intégration avec Inertia.js
  • Partage des props Laravel vers Vue
  • Layouts, navigation, gestion des erreurs
  • Refonte des vues Login/Register avec Vue

Module 4 : Tailwind CSS et UI Responsive

Semaine 4

  • Installation de Tailwind via Laravel Mix ou Vite
  • Structure responsive avec Flex et Grid
  • Utilisation des utilitaires Tailwind
  • Composants UI : cards, forms, boutons, alertes
  • Thématisation et responsive design
  • Transitions, animations, dark mode
