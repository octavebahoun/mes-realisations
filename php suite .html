Partie 1 : Bases du langage PHP

Chapitre 1 : Introduction au PHP

PHP est un langage de programmation principalement utilisé pour créer des sites web dynamiques. Il s’exécute côté serveur et peut générer du contenu HTML, interagir avec des bases de données, gérer des sessions, etc.

Mémo : Pour commencer, installe un serveur local comme XAMPP ou WAMP qui contient Apache, PHP et MySQL. Active le serveur Apache et place tes fichiers PHP dans le dossier htdocs (XAMPP) ou www (WAMP).
Astuce développeur : En développement, active l’affichage des erreurs avec error_reporting(E_ALL); ini_set('display_errors', 1); au début de tes scripts PHP pour faciliter le debug.
Exercice : Crée un fichier index.php contenant la ligne <?php echo "Hello, world!"; ?>. Ouvre-le dans ton navigateur via localhost/index.php et vérifie que le message s’affiche.

Chapitre 2 : Syntaxe de base

En PHP, les variables commencent toujours par $. Le langage est faiblement typé, donc les types sont attribués automatiquement.

// Exemple de variables
$nom = "Octave";
$age = 23;
$estEtudiant = true;

echo "Je m'appelle $nom et j'ai $age ans."; // Interpolation
Mémo : Les principaux types en PHP sont string, int, float, bool, array, object, et null.
Exercice : Déclare des variables pour ton prénom, ton âge, et ton hobby. Affiche une phrase utilisant ces variables.

Chapitre 3 : Structures de contrôle

Les structures conditionnelles permettent d’exécuter des blocs de code selon des conditions :

if ($age >= 18) {
    echo "Adulte";
} else {
    echo "Mineur";
}

Les boucles permettent de répéter du code :

for ($i = 0; $i < 5; $i++) {
    echo $i;
}
Astuce : Utilise foreach pour parcourir facilement les tableaux.
Exercice : Crée un tableau de fruits puis affiche-les un par un avec une boucle foreach.

Chapitre 4 : Fonctions

Les fonctions permettent de regrouper du code réutilisable :

function saluer($nom) {
    return "Bonjour, $nom !";
}

echo saluer("Octave");
Mémo : Les fonctions peuvent avoir des paramètres, retourner des valeurs, et utiliser des paramètres optionnels.
Exercice : Crée une fonction qui calcule la somme de deux nombres et teste-la avec différents arguments.

Chapitre 5 : Tableaux

PHP supporte plusieurs types de tableaux : indexés, associatifs, multidimensionnels.

$fruits = ["pomme", "banane", "cerise"];
$infos = ["nom" => "Octave", "age" => 23];
Astuce : Utilise array_map et array_filter pour manipuler les tableaux plus facilement.
Exercice : Crée un tableau associatif avec tes amis et leurs âges, puis affiche ceux qui ont plus de 20 ans.

Chapitre 6 : Superglobales et entrées utilisateur

Les superglobales permettent d’accéder aux données envoyées par le client et de gérer les sessions :

  • $_GET : données passées dans l’URL
  • $_POST : données de formulaire envoyées en POST
  • $_SESSION : variables de session côté serveur
  • $_COOKIE : cookies stockés sur le client
Mémo : Toujours filtrer et valider les données entrantes pour éviter les failles de sécurité.
Exercice : Crée un formulaire HTML simple qui envoie un nom en POST et affiche-le en PHP.

Partie 2 : Programmation Orientée Objet (POO)

Chapitre 7 : Les bases de la POO

La POO permet de modéliser des objets avec des propriétés et des méthodes :

class Utilisateur {
    public $nom;
    public $age;

    public function __construct($nom, $age) {
        $this->nom = $nom;
        $this->age = $age;
    }

    public function saluer() {
        return "Bonjour, je suis " . $this->nom;
    }
}

$user = new Utilisateur("Octave", 23);
echo $user->saluer();
Mémo : Le mot-clé $this fait référence à l'objet courant.
Astuce : Utilise la visibilité private pour protéger les propriétés et créer des getters/setters.
Exercice : Crée une classe Voiture avec des propriétés marque et couleur, une méthode pour afficher ses infos.

Chapitre 8 : Encapsulation et héritage

On peut contrôler l'accès aux propriétés avec public, protected et private. L’héritage permet de créer des classes enfants :

class Admin extends Utilisateur {
    public function supprimerUtilisateur() {
        return "Utilisateur supprimé";
    }
}

$admin = new Admin("Alice", 30);
echo $admin->saluer();
echo $admin->supprimerUtilisateur();
Mémo : parent:: permet d'appeler les méthodes de la classe parente.
Exercice : Crée une classe Employe qui hérite de Utilisateur et ajoute une propriété poste.

Chapitre 9 : Espaces de noms et autoloading

Les namespaces évitent les collisions de noms dans les projets importants :

namespace MonProjet\Models;

class Utilisateur {
    // ...
}
Astuce : Utilise Composer pour autoloader automatiquement tes classes selon la norme PSR-4.
Exercice : Organise un mini-projet en dossiers avec namespaces et utilise un autoloader simple.

Partie 3 : Développement web dynamique avec PHP

Chapitre 10 : Formulaires HTML et traitement PHP

Le formulaire HTML envoie des données à un script PHP :

<form method="POST" action="traitement.php">
    <input type="text" name="prenom" />
    <input type="submit" value="Envoyer" />
</form>

En PHP, on récupère les données avec $_POST :

$prenom = $_POST['prenom'] ?? '';
echo "Bonjour, $prenom";
Mémo : Toujours valider et nettoyer les données reçues.
Exercice : Crée un formulaire qui demande l’âge, puis affiche « Majeur » ou « Mineur » selon la réponse.

Chapitre 11 : Sessions, cookies et gestion d’état

Les sessions permettent de conserver des données entre les pages :

session_start();
$_SESSION['user'] = "Octave";

Les cookies sont stockés côté client :

setcookie("prenom", "Octave", time()+3600);
Astuce : Regénère l’ID de session après connexion pour sécuriser (session_regenerate_id()).
Exercice : Implémente un mini système de login/logout simple avec sessions.

Chapitre 12 : Inclusion de fichiers

Pour structurer ton projet, inclure des fichiers est très utile :

include 'header.php';
include_once 'footer.php';
Mémo : require provoque une erreur fatale si le fichier est manquant, include seulement un avertissement.
Exercice : Crée un site simple avec un header et footer inclus.

Chapitre 13 : Base de données MySQL et PDO

Connexion sécurisée avec PDO :

try {
    $pdo = new PDO("mysql:host=localhost;dbname=ma_base", "root", "");
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
} catch(PDOException $e) {
    die("Erreur : " . $e->getMessage());
}

Requête préparée :

$stmt = $pdo->prepare("SELECT * FROM utilisateurs WHERE email = :email");
$stmt->execute(['email' => $email]);
$user = $stmt->fetch();
Mémo : Toujours utiliser des requêtes préparées pour éviter les injections SQL.
Exercice : Crée une table utilisateurs et insère des données via PHP.

Chapitre 14 : Authentification simple

Stockage sécurisé des mots de passe avec password_hash() et vérification avec password_verify() :

$hash = password_hash($motdepasse, PASSWORD_DEFAULT);

if (password_verify($motdepasse_saisi, $hash)) {
    echo "Connexion réussie";
}
Astuce : Ne jamais stocker les mots de passe en clair.
Exercice guidé : Crée un mini système d’inscription et connexion avec sessions.
