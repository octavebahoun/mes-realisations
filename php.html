Cours PHP complet pour débutant – Partie 1 : Bases du langage PHP

Chapitre 1 : Introduction au PHP

PHP est un langage de programmation principalement utilisé pour créer des sites web dynamiques. Il s’exécute côté serveur et peut générer du contenu HTML, interagir avec des bases de données, gérer des sessions, etc.

Mémo : Pour commencer, installe un serveur local comme XAMPP ou WAMP qui contient Apache, PHP et MySQL. Active le serveur Apache et place tes fichiers PHP dans le dossier htdocs (XAMPP) ou www (WAMP).
Astuce développeur : En développement, active l’affichage des erreurs avec error_reporting(E_ALL); ini_set('display_errors', 1); au début de tes scripts PHP pour faciliter le debug.
Exercice : Crée un fichier index.php contenant la ligne <?php echo "Hello, world!"; ?>. Ouvre-le dans ton navigateur via localhost/index.php et vérifie que le message s’affiche.

Chapitre 2 : Syntaxe de base

En PHP, les variables commencent toujours par $. Le langage est faiblement typé, donc les types sont attribués automatiquement.

// Exemple de variables
$nom = "Octave";
$age = 23;
$estEtudiant = true;

echo "Je m'appelle $nom et j'ai $age ans."; // Interpolation
Mémo : Les principaux types en PHP sont string, int, float, bool, array, object, et null.
Exercice : Déclare des variables pour ton prénom, ton âge, et ton hobby. Affiche une phrase utilisant ces variables.
