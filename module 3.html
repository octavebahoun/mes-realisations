Planning Laravel + Vue.js + Tailwind (8 semaines)

Module 1 : Laravel Basics

  • Installation via Laravel Installer ou Composer
  • Structure d’un projet Laravel (routes, contrôleurs, vues, modèles)
  • Artisan CLI : les commandes utiles
  • Migration et Eloquent ORM (modèles + relations)
  • Premiers contrôleurs et affichage de données
  • Utilisation de Tinker pour tester en direct

Module 2 : Authentification, Middleware et Sécurité

  • Installation de Laravel Breeze ou Laravel Jetstream (Inertia ou Livewire)
  • Création des utilisateurs, pages login/register
  • Authentification personnalisée : middleware, guards
  • Rôles et permissions simples (admin, user)
  • Protection des routes (auth, verified, admin...)
  • Validation des formulaires et messages d’erreurs
  • Hashage de mots de passe et sécurité des données
  • Gestion des erreurs (try/catch, abort, pages 403/404/500)

Module 3 : Vue.js Basics avec Inertia

  • Introduction à Inertia.js : liaison Laravel ↔ Vue
  • Création de composants Vue : structure, props, slots
  • Passage de données Laravel → Vue via Inertia
  • Navigation côté client avec Inertia
  • Utilisation de layouts avec Vue/Inertia
  • Refactorisation de pages Laravel → Vue
  • Interactivité de base : v-model, événements, conditions
  • Communication entre composants (émission, réception)
