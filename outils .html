Outils Dev Fullstack Laravel + Vue + Tailwind

1. Environnement de développement local

  • PHP (≥ 8.x) : Laravel nécessite PHP 8 ou supérieur.
  • Composer : gestionnaire de dépendances PHP.
  • Node.js & npm/yarn : pour gérer les dépendances frontend et compiler les assets.
  • Serveur local :
    • Laravel Sail (Docker, recommandé)
    • Sinon XAMPP / MAMP / WAMP (moins conseillé)

2. Framework & librairies

  • Laravel Framework (backend PHP)
  • Vue.js (framework frontend, version 3 recommandée)
  • Tailwind CSS (framework CSS utilitaire)
  • Inertia.js (optionnel, pour SPA Laravel + Vue sans API REST séparée)
  • Axios ou fetch (pour requêtes AJAX)

3. Éditeur de code / IDE

  • Visual Studio Code (très populaire et gratuit)
  • Extensions recommandées :
    • Laravel Blade Snippets
    • PHP Intelephense
    • Volar ou Vetur (pour Vue 3)
    • Tailwind CSS IntelliSense
    • ESLint / Prettier
    • GitLens

4. Base de données

  • MySQL / MariaDB (classique avec Laravel)
  • Ou PostgreSQL
  • Clients pour gérer la base :
    • phpMyAdmin (interface web)
    • TablePlus / DBeaver / HeidiSQL (clients desktop)

5. Gestion de versions

  • Git (contrôle de version)
  • Plateformes : GitHub, GitLab, Bitbucket

6. Outils frontend build / compilation

  • Laravel Mix (webpack, par défaut)
  • Vite (plus moderne, recommandé Laravel 9+)
  • Commandes npm courantes :
    • npm install / yarn — installer dépendances
    • npm run dev — build dev avec watch
    • npm run build — build production

7. Debugging & tests

  • Laravel Telescope (debug backend Laravel)
  • Laravel Debugbar (barre debug dans navigateur)
  • Vue Devtools (extension navigateur pour Vue)
  • Tests automatisés :
    • PHPUnit (backend)
    • Jest / Cypress (frontend)

8. Déploiement

  • Serveur web : Apache ou Nginx + PHP-FPM
  • Laravel Forge (outil de déploiement simplifié)
  • Envoyer (outil Laravel de déploiement)
  • CI/CD : GitHub Actions, GitLab CI
  • Hébergement : DigitalOcean, Linode, AWS, Hetzner...

9. Autres outils utiles

  • Postman / Insomnia (tester API)
  • Docker (environnement dev stable)
  • Tinkerwell (tester code Laravel en live)
  • npm-run-all (gérer scripts npm en parallèle)

Résumé minimal pour commencer

  • PHP 8 + Composer
  • Laravel via Composer
  • Node.js + npm/yarn
  • VS Code + extensions Blade, Vue, Tailwind
  • MySQL + client DB
  • Git + GitHub
  • Laravel Mix ou Vite pour builder frontend
  • Vue Devtools
  • Postman
