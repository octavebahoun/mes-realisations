Planning Complet — Laravel + Vue.js + Inertia + Tailwind (2 mois)

Module 1 — Laravel Basics

  • Installation avec Laravel Installer ou Composer
  • Structure des dossiers
  • Routing, Contrôleurs, Vues Blade
  • Migrations, seeders, Eloquent ORM
  • Requêtes HTTP & logique des contrôleurs
Durée : 1 semaine

Module 2 — Authentification, Middleware et Sécurité

  • Installation Laravel Breeze (ou Jetstream)
  • Inscription, Connexion, Déconnexion
  • Middleware : auth, guest, custom
  • Protection des routes et CSRF
  • Validation des formulaires & erreurs
Durée : 1 semaine

Module 3 — Vue.js Basics avec Inertia

  • Installation Inertia.js + Vue 3
  • Pages Inertia, Layouts, Navigation
  • Props, data, events, composants
  • Refactoring de Blade → Vue
  • Organisation du frontend
Durée : 1 semaine

Module 4 — Tailwind CSS et UI Responsive

  • Installation et configuration de Tailwind CSS
  • Grilles, flexbox, spacing, couleurs
  • Composants UI réutilisables
  • Responsive design avec breakpoints
  • Dark mode, transitions, hover
Durée : 1 semaine

Module 5 — CRUD Complet avec Laravel + Vue + Inertia

  • Créer une ressource : routes, contrôleur, modèle, migration
  • Affichage d’une liste dans Vue avec Inertia
  • Création, modification, suppression avec formulaire Vue
  • Validation côté Laravel et côté Vue
  • Notifications, toasts, redirections
Durée : 1 semaine
